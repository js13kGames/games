Join the hamster uprising!

Drive around as a hamster shooting humans! It can't take that long, can it?

CONTROLS:
  W / UP: Drive forwards
  S / DOWN: Reverse
  A-D / LEFT-RIGHT: Turn left or right
  SHIFT: Vent
  SPACE / CLICK: Shoot
  Mouse movement: Aim
  P / ESCAPE: Pause
