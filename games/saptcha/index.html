Saptcha

Slightly Automated Public Turing test to tell Computers and Humans Apart

Imagine that you are at the back of the Captcha system. The programmers didn't have time to code a real artificial intelligence so now you have to match the pictures with the given name to help websites test if the user is a Human or a malicious Robot.

How to play

Select all images that look like the animal given by the name and than click verify.
For each correct answer you earn 35 points and you lose 25 points for each mistake.
Earn points to unlock new types of animals.
