The Matrix is under attack! Obliterate all Hackers and get rid of the red glitches as they import dangerous drones.

Play with:
[W,A,S,D] - Move
[Shift] - Special
[E] - Grenade
[Mouse] - Aim and shoot
[1,2,3,4,5] or [X,Y] - Change weapons
