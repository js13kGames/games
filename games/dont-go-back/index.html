    Help

    "Don't Go Back" is a maze game. Use Play menu to load main game. Use arrow keys to control player. You have to go to the green spot to win. Field of view is restricted. Visited places are dangerous. You will lose some life if you will visit them again. Make your own levels using Creator's Mode.

    Keys usage:

    Arrows - movements.

    A/S/D/W - create/delete Left/Down/Right/Top wall of the current cell.

    Z - set level's start cell.

    X - set level's end cell.
